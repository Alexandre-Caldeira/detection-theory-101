Contents
+++++++++

Data set can be loaded into matlab using

load Subject_ABR_Data.mat

Contains Subject ABR data from the 0, 10, 20, 30, 40, and 50 dB SL conditions
Data is saved in cell format. 


Geographic location of data collection: University of Southampton, U.K.

Related projects:
EPSRC, grant No. EP/M026728/1
S Bell 2015 - Personalised fitting & evaluation of hearing aids with EEG responses
